# Naciye Duygu — Kişisel Portfolyo

Saf HTML, CSS ve JavaScript ile geliştirilmiş, modern ve etkileşimli kişisel portfolyo sitesi.

---

## 📁 Proje Yapısı

```
web_portfolio_naciyeduygu/
├── index.html          # Ana sayfa
├── README.md           # Bu dosya
├── pages/
│   ├── about.html      # Hakkında sayfası
│   ├── contact.html    # İletişim sayfası
│   └── project.html    # Projeler sayfası (Bu dosya henüz boş)
└── assets/
    ├── css/
    │   └── style.css   # Ana stil dosyası
    ├── img/            # Resimler ve diğer medya dosyaları
    └── js/
        └── app.js      # Ana JavaScript dosyası
```

---

## 🚀 Kullanılan Teknolojiler

* HTML5: Web sayfasının yapısal iskeletini oluşturur.
* CSS3: Modern tasarım ve stil için (CSS Variables, Flexbox, Grid).
* Vanilla JavaScript (ES6+): Etkileşimli özellikler ve animasyonlar.
* Google Fonts: Playfair Display ve JetBrains Mono fontları.

---

## ⚙️ Kurulum

Projeyi yerel makinenizde çalıştırmak için herhangi bir özel kurulum gerekmez.

**Depoyu Klonlayın**

```
git clone https://github.com/web-projesi-2026/web_portfolio_naciyeduygu.git
```

**Proje Dizinine Gidin**

```
cd web_portfolio_naciyeduygu
```

**Tarayıcınızda Açın**

```
index.html
```

---

## ✨ Özellikler

* Tam Duyarlı Tasarım
* Özel İmleç
* Animasyonlu Arka Plan
* Hamburger Menü
* Kaydırma Animasyonları (Scroll Reveal)
* Sayıcı Animasyonları ("Tamamlanan Proje" ve "Uyku Sevgisi")
* Çok Adımlı İletişim Formu
* Proje Kartı Parlama Efekti
* Aktif Sayfa Vurgusu (Navigasyon)
